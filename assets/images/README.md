# Image Assets

This directory contains all image assets for the portfolio website.

## Required Images

### Hero Section
- `hero-bg.jpg` - Background image for hero section (blurred)
  - Recommended size: 1920x1080px
  - Will be blurred with 18.5px blur and 50% opacity

### Spotlight Section
- `spotlight-project.jpg` - Featured project showcase
  - Recommended size: 800x600px

### Project Cards
- `project-1.jpg` through `project-6.jpg` - Project thumbnails
  - Recommended size: 600x400px each
  - Should showcase individual project work

## Image Optimization Tips

1. **Format**: Use JPG for photos, PNG for graphics with transparency
2. **Compression**: Optimize all images before uploading
   - Tools: TinyPNG, ImageOptim, Squoosh
3. **Responsive**: Consider using WebP format with JPG fallback
4. **Sizing**: Keep file sizes under 200KB per image
5. **Alt Text**: All images have descriptive alt text in HTML
